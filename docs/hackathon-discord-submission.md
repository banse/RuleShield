Submission for the Hermes Agent Hackathon:

RuleShield is an optimization layer for Hermes Agent. It sits in front of the upstream LLM, intercepts repetitive prompts, evaluates candidate rules in shadow mode, and improves rule confidence through feedback over time.

The project focuses on reducing repetitive LLM spend safely and transparently, while also creating a path toward optimizing recurring Hermes workflows and cron-style automations.

The demo shows:
- Hermes running through RuleShield
- shadow mode for safe rule validation
- live trigger and savings visibility
- the feedback loop in action
