# Hackathon ElevenLabs Voice Guide

## Ziel

Schneller TTS-Pfad fuer das Hackathon-Video, falls OpenAI API Billing blockiert ist.

## Empfohlener Stil

Keine direkte Imitation eines Filmcharakters. Stattdessen:

- maennlich
- laid-back
- warm
- low-energy but confident
- leicht rau
- ruhig
- unaufgeregt
- kein Werbesprecher

## Voice Prompt Option 1

```text
Male voice, relaxed and conversational, warm, low-energy but confident, slightly rough texture, mellow and easygoing, dry understated humor, natural American tone, slow-medium pace, human and unpolished, not announcer-like, not too dramatic, calm and casually intelligent.
```

## Voice Prompt Option 2

```text
Male voice, very laid-back and mellow, warm and slightly sleepy, gently raspy, calm and effortless, casual American delivery, unhurried pace, understated humor, sounds relaxed, thoughtful, and completely unbothered.
```

## Voice Prompt Option 3

```text
Male voice, calm, warm, grounded, conversational, slightly rough, low-energy but clear, natural American tone, measured pace, understated and human, not polished, not promotional, not theatrical.
```

## Empfehlung

Test-Reihenfolge:

1. Option 1
2. Option 3
3. Option 2

## Workflow

1. Einen natuerlichen maennlichen Voice-Kandidaten waehlen
2. Den Voice Prompt in die Description / Style-Eingabe uebernehmen, falls verfuegbar
3. Die 4 Voiceover-Bloecke einzeln rendern
4. Nur problematische Bloecke neu erzeugen
5. Danach alles im Schnittprogramm zusammensetzen
